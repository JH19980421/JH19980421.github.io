

## 1. 변수

```java
int a = 100; // 변수 선언과 초기화 (100은 리터럴)
```

### 1.1 변수 타입

- 논리형 : boolean  
- 정수형 : int, long  
- 실수형 : double  
- 문자열 : String  

#### 클래스 변수 (static 변수)

- static 키워드가 붙은 변수
- 모든 인스턴스가 공통된 저장공간을 공유
- 클래스 로딩 시 생성, 프로그램 종료 시까지 유지

```java
static int count;
```

#### 인스턴스 변수

```java
int age;
```

### 1.2 변수 명명 규칙

- 변수명 : 소문자 시작
- 클래스명 : 대문자 시작
- 카멜 케이스 사용
- 상수 : 대문자 + 언더바

```java
String orderDetail;
class LeftPerson;
final int USER_COUNT = 10;
```

## 2. 연산자

### 2.1 문자열 덧셈

```java
String a = "ab";
String b = "cde";
System.out.println(a + b); // abcde
```

### 2.2 ++a 와 a++ 차이

```java
int a = 1, b = 0;
b = ++a; // a=2, b=2
```

```java
int a = 1, b = 0;
b = a++; // a=2, b=1
```

## 3. Java 14 switch 표현식

```java
int grade = 2;

int coupon = switch (grade) {
    case 1 -> 1000;
    case 2 -> 2000;
    case 3 -> 3000;
    default -> 0;
};

System.out.println(coupon); // 2000
```

## 5. Scanner


### 5.1 Scanner vs BufferedReader

| 구분 | Scanner | BufferedReader |
|---|---|---|
| 입력 | 타입별 가능 | 문자열만 |
| 형변환 | 자동 | 수동 |
| 예외 | 선택 | 필수 |
| 성능 | 느림 | 빠름 |
| 동기화 | X | O |

*동기화 :　한 쓰레드에서 메소드가 실행될 때, 다른 쓰레드가 입력, 수정 등의 행위를 할 수 없게 제한함.

## 6. 메소드 오버로딩

```java
public class Calculator {

    public static int addNumber(int a, int b) {
        return a + b;
    }

    public static double addNumber(double a, double b) {
        return a + b;
    }

    public static float addNumber(float a, float b) {
        return a + b;
    }
}
```
