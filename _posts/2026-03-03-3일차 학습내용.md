### 생성자

생성자란 인스턴스가 생성될 때 호출되는 인스턴스 초기화(=인스턴스 변수 초기화) 메소드. 

주로 인스터스 변수 초기화 작업을 위해 수행되며, 인스턴스가 생성될 때 실행되어야 할 작업을 위해서도 사용된다. 생성자 또한 메소드이기 때문에 오버 로딩으로 여러 생성자를 생성할 수 있다.

생성자를 생성하기 위해서는 생성자의 이름과 클래스의 이름이 같아야하고 리턴 값이 없어야한다.  
<br>
<br>

```java
class Student{
    int grade;
    int studentNo;

    public Student(int grade, int studentNo){ 
    	this.grade = grade;
      this.studentNo = studentNo;
    }
}
```

### 제어자  
<br>
<br>

제어자는 클래스, 변수 또는 메소드의 선언부에 함께 사용되어 추가적인 의미를 부여. 

접근 제어자와 그 외 나머지로 구분된다. 접근 제어자를 제외하면, 여러 개의 제어자를 혼합해서 사용이 가능하다.

접근 제어자 : public, protected, default, private (이 중에서 하나만 선택해서 사용 가능)

그 외 나머지 : static, final, abstract 등등

### JVM

<br>


JVM은 시스템으로부터 프로그램을 수행하는데 필요한 메모리를 효율적으로 할당 받기 위해, 여러 영역으로 관리한다.

- 메소드 영역 - 클래스 변수, 메소드, static 영역
- 스택 영역 - 메소드 작업에 필요한 메모리 공간 제공
- 힙 - 인스턴스 변수, 배열 생성

```java
class User {
    String name;
    int count;
    public User(String name, int count) {
        this.name = name;
        this.count = count;
    }
    void addCount() {this.count++;}
    void printUser() {
        System.out.println("유저 : " + this.name + ", 카운트 : " + this.count);
    }
}

class Shop {
    public static void main(String[] args) {
        User user = new User("아무개", 1);
        user.printUser();
        User user2 = new User("아무개2", 1);
        user.addCount();
        user.printUser();
        user2.printUser();
    }
}

```

### 상속

<br>

상속이란 기존에 작성된 클래스를 사용하여 새로운 클래스를 작성하는 것이다. 

상속받을 클래스를 extends와 함께 사용하여 상속받은 클래스의 멤버변수와 메소드를 사용하면서 새 기능을 추가할 수 있다. 상속하는 클래스를 부모 클래스, 상속받는 클래스를 자식 클래스라고 한다.

자식 클래스는 부모 클래스로 부터 변수와 메소드를 물려받기 때문에 부모 클래스의 변수와 메소드를 사용할 수 있다.

다만 생성자와 초기화 블럭은 상속되지 않고 오로지 멤버만 상속된다.

또한, 자식 클래스는 여러 개의 부모 클래스를 상속 받을 수 없고 오로지 하나의 부모 클래스만을 상속 받을 수 있다.

<br>
<br>

### 오버라이딩

<br>


부모 클래스로부터 상속받은 메소드의 내용을 자식 클래스에 맞게 변경하는 것.

<br>
<br>

### super & super()

<br>


super는 자식 클래스에서 부모 클래스로부터 상속받은 멤버를 참조하는데 사용된다. 

super()는 부모 클래스의 생성자를 호출하는데 사용된다.

자식 클래스의 인스턴스를 생성하려면, 부모 클래스 인스턴스가 먼저 생성되어야 하기 때문에, 자식 클래스의 생성자에 super()를 정의하여 먼저 부모 인스턴스를 생성한다.  

표기하지 않으면 컴파일러가 자동적으로 super();를 추가한다.

<br>
<br>

```java
class Student{
    int grade;
    String subject;
    public Student(int grade, String subject) {
        this.grade = grade;
        this.subject = subject;
    }

    void printGrade() {System.out.println("잠시 후에 확인해주세요.");
    }

    void printSubject() {System.out.println(subject);}
}

class Attendance extends Student{
    boolean check;

    public Attendance(int grade, String subject, boolean check) {
        super(grade, subject);
        this.check = check;
    }

    void getCheck(){
        if(check) {
            System.out.println("출석확인.");
        } else {
            System.out.println("출석확인불가.");
        }
    }
    void printGrade() {System.out.println("학년 : " + grade);
    } // 오버라이딩
}

public class School{

    public static void main(String[] args){
        // TODO Auto-generated method stub
        Attendance attendance = new Attendance(1,"아무개", true);
        
        attendance.getCheck(); // 출석확인
        attendance.printSubject(); // 아무개
        attendance.printGrade(); // 학년 1
    }
}

```
